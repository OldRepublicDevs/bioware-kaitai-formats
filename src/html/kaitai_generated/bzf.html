Bzf format specification

Type: Bzf

BZF (BioWare Zipped File) files are LZMA-compressed BIF files used primarily in iOS (and maybe Android) ports of KotOR. The BZF header contains "BZF " + "V1.0", followed by LZMA-compressed BIF data. Decompression reveals a standard BIF structure. Format Structure: - Header (8 bytes): File type signature "BZF " and version "V1.0" - Compressed Data: LZMA-compressed BIF file data After decompression, the data follows the standard BIF format structure. References: - https://github.com/OldRepublicDevs/PyKotor/wiki/BIF-File-Format.md - BZF compression section - BIF.ksy - Standard BIF format (decompressed BZF data matches this)

OffsetSizeIDTypeNote
0 ... file_type str(ASCII) File type signature. Must be "BZF " for compressed BIF files.
4 ... version str(ASCII) File format version. Always "V1.0" for BZF files.
8 ... compressed_data u1 LZMA-compressed BIF file data. This data must be decompressed using LZMA algorithm to obtain the standard BIF structure. After decompression, the data can be parsed using the BIF format definition.
