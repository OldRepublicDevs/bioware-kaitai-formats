Uts format specification

Type: Uts

UTS (Sound Template) files define ambient sound sources and properties. This format inherits the complete GFF structure from gff.ksy. UTS Root Struct Fields: - TemplateResRef (ResRef): Blueprint identifier - Tag (String): Instance identifier - LocalizedName (LocalizedString): Sound name - Sounds (List): Sound file references - Volume, MaxDistance, MinDistance: Audio properties - Looping, Random, Positional: Sound behavior
