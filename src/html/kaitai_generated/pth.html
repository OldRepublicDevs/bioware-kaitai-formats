Pth format specification

Type: Pth

PTH (Path/Pathfinding) files store pathfinding waypoint graphs for AI navigation. This format inherits the complete GFF structure from gff.ksy. PTH Root Struct Fields: - Path_Points (List): Waypoint positions (X, Y coordinates) - Connections (List): Connections between waypoints References: - https://github.com/OldRepublicDevs/PyKotor/wiki/GFF-PTH.md - https://github.com/OldRepublicDevs/PyKotor/wiki/GFF-File-Format.md - https://github.com/OldRepublicDevs/PyKotor/blob/master/Libraries/PyKotor/src/pykotor/resource/generics/pth.py
